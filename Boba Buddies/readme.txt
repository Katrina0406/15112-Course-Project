- Install NLTK :
  https://www.nltk.org/install.html
  
- Install punkt:
  Unzip 'punkt' package in the folder

- Install text2emotion module: pip install text2emotion

- Run the Dashboard.py file in the folder
  * click 'new diary' to enter the diary page
  * type in your diary and click 'get boba' to submit
  * a boba drink will be recommended by your mood according to the diary
  * click 'Try Again!" to reenter a new diary

- Note: If text2emotion doesn't work, run it in JupyterLab instead.

- Enjoy!!:)
